비선형구조 : i번째 원소를 탐색한 후 다음으로 탐색할 i+1번째 원소가 여러개인것 ex) 트리, 그래프 등
비선형탐색 : 비선형구조의 모든 정점을 탐색 <- stack, queue와 같은 자료구조 사용

그래프의 구현 : 
	1) 인접행렬 (adjacency matrix) : 2차원 행렬에서 index에 정점을 넣고 서로 연결된 간선이 있으면 1, 없으면 0을 넣음 / 간선에 가중치가 있다면 가중치 값을 넣음 -> O(nm)
	2) 인접리스트 (adjacency list) : 각 정점에 대해 연결된 정점을 리스트로 표현 ; 연결되지 않은 부분은 나타내지 않으므로 효율적 -> O(n+m)

트리 = 회로(loop)가 없는 그래프

1. 깊이우선탐색 (Depth First Search; DFS) : 모든 정점을 깊이 우선으로 탐색, 한번 끝까지 내려갔다면 다시 올라오는 백트랙(backtrack)과정 필요
2. 너비우선탐색 (Breadth First Search; BFS)
